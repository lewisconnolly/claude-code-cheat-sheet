# Claude Code Cheat Sheet

A quick reference guide for [Claude Code](https://claude.ai/code) - Anthropic's official CLI for Claude.

**[View the Cheat Sheet](https://lewisconnolly.github.io/claude-code-cheat-sheet/)**

## Contents

- **Essential Shortcuts** - Keyboard shortcuts for auto-accept mode, planning mode, thinking toggle, and more
- **Input Shortcuts** - Prompt editing shortcuts for multiline input, text deletion, cursor navigation, and history
- **Initialize Project** - Using `/init` to create CLAUDE.md files
- **Config File Locations** - Where settings, memory files, and custom commands are stored
- **Thinking Modes** - Keywords to control reasoning depth (`think`, `megathink`, `ultrathink`)
- **Custom Slash Commands** - Creating project and global custom commands with `.md` files
- **Built-in Tools** - Reference for Read, Write, Edit, Bash, Glob, Grep, Task, WebFetch, and more
- **Hook Types** - PreToolUse, PostToolUse, UserPromptSubmit, and other hook events
- **MCP Servers** - Adding and managing Model Context Protocol servers
- **Python SDK** - Using `claude-agent-sdk` for programmatic access
- **Hook Examples** - Logging tool inputs with hooks

## About

Single-page static HTML reference with a dark theme. No build system or dependencies required.
